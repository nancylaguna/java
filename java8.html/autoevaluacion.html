Ejercicios de autoevaluación

    8.1 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué

  1. El caso default es requerido en la instrucción de selección switch. Falso: El caso default es opcional. Si no se necesita una acción predeterminada, entonces no hay necesidad de un caso default.
  2. La instrucción break es requerida en el último caso de una instrucción de selección switch. Falso. La instrucción break se utiliza para salir de switch. La instrucción break no se requiere para el último caso de switch.
  3. La expresión (x > y && a < b) es verdadera si x > y es verdadera, o si a < b es verdadera. Falso. Ambas expresiones relacionales deben ser verdaderas para que toda la expresión sea verdadera cuando se utilice el operador &&.
  4. Una expresión que contiene el operador || es verdadera si uno o ambos de sus operadores son verdaderos. Verdadero

    8.3 Encuentre el error en cada uno de los siguientes segmentos de código, y explique cómo corregirlo:

  1. x = 1; while(x <= 10); ++x; } Error: El punto y coma después del encabezado while provoca un ciclo infinito; además falta una llave izquierda. Correción: Reemplazar el punto y coma por una llave izquierda ({), o eliminar tanto el punto y coma (;) como la llave derecha (}).
  2. switch(n){ case 1: document.writeln("El número es 1"); case 2: document.writeln("El número es 2"); break; default: document.writeln("El número no es 1 ni 2"); break; } Error: Falta una instrucción break en las instrucciones del primer case. Correción: Agregue una instrucción break al final de las instrucciones para el primer case. Cabe mencionar que esta omisión no es necesariamente un error, si desea que la instrucción del case 2: se ejecute siempre que lo haga la instrucción del case 1:.
  3. El siguiente código debe imprimir los valores 1 a 10: n = 1; while (n < 10) document.writeln(n++); Error: Se está utilizando un operador relacional inadecuado en la condición de continuación de la instrucción de repetición while. Correción: Use <= en vez de <, o cambie el 10 a 11.
