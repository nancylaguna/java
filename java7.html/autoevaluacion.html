Ejercicios de autoevaluación

    7.1 Complete los siguientes enunciados:

  1. Todas las secuencias de comandos pueden escribirse en términos de tres tipos de estructuras de control: secuencia, selección y repetición.
  2. La instrucción de selección doble if....else se utiliza para ejecutar una acción cuando una condición es verdadera, y otra acción cuando esa condición es falsa.
  3. Al proceso de repetir un conjunto de instrucciones un número específico de veces se le llama repetición controlada por contador (o definida).
  4. Cuando no se sabe de antemano cuántas veces se repetirá un conjunto de instrucciones, se puede usar un valor centinela (o un de señal, de bandera o de prueba) para terminar la repetición.

    7.2 Escriba cuatro instrucciones de JavaScript que sumen 1, cada una, a la variable x, que contiene un número.

  1. x = x + 1;
  2. x += 1;
  3. ++x;
  4. x++;

    7.3 Escriba instrucciones en JavaScript para realizar cada una de las siguientes tareas:

  1. Asignar la suma de x y y a z, e incrementar el valor de x en 1 después del cálculo. Usar sólo una insttrucción. z = x++ + y;
  2. Evaluar si el valor de la variable cuenta es mayor que 10. De ser así, imprimir "Cuenta es mayor que 10". if (cuenta > 10) document.writeln("Cuenta es mayor que 10");
  3. Decrementar la variable x en 1, luego restarla a la variable total. Usar sólo una instrucción. total -= --x;
  4. Calcular el residuo después de dividir q entre divisor y asignar el resultado a q. Escriba esta instrucción de dos maneras distintas. q %= divisor; q = q % divisor;

    7.4 Escriba una instrucción en JavaScript para realizar cada una de las siguientes tareas:

  1. Declarar las variables suma y x. var suma, x;
  2. Asignar 1 a la variable x. x = 1;
  3. Asignar 0 a la variable suma. suma = 0;
  4. Sumar la variable x a suma y asignar el resultado a la variable suma. suma += x; o suma = suma + x;
  5. Imprimir "La suma es: ", seguido del valor de la variable suma. document.writeln("La suma es: ", + suma);

    7.8 ¿Qué está mal en la siguiente instrucción de repetición while?

  1. while (z >= 0) suma += z; El valor de la variable z nunca se cambia en el cuaerpo de la instrucción while. Por lo tanto, si la condición de continuación de ciclo (z >= 0) es verdadera, se crea un ciclo infinito. Para evitar que ocurra un ciclo infinito, z debe decrementarse de manera que en un momento dado se vuelva menor que 0.
